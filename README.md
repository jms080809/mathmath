## #Usage

Deploy 하려면

```bash
// 이 코드는 dist 에서 서버를 실행시킬 것입니다!
npm install
npm run build
npm start
```

개발 상태에서

```bash
// 실제 서버에서 이것을 사용하는 것은 매우 위험!(근데 어짜피 깃 클론해서 설치한 후 사용해야 돼서 별 의미는 없음 ㅋ)
npm install
npm run dev
```
